# 睿思  RAG  知识问答平台

#### 介绍
一个面向本地私有知识库的 Agentic RAG 智能问答系统。不同于传统 RAG 所采用的“检索→生成”单链路架构，
本系统引入了多工具编排、对话记忆管理以及自我反思机制，使大模型能够主动评估检索质量、调用外部工具以
补充知识，并跨会话理解上下文，从而实现“像人一样思考”的问答体验。
